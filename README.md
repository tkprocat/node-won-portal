<h1>Installation:</h1>

git clone https://github.com/tkprocat/node-won-portal.git

cd node-won-portal 

npm install

<h1>Configuration:</h1>

Open config.js

Change the line "name: 'Example', ip: '192.168.2.1', mac: 'AA:BB:CC:DD:EE:FF'" to appropiate values.

Change the default password (default password is secret).

<h1>Run:</h1>

npm start
